# Workforce Planning & Utilization Analysis (Power BI)

## Project Overview
This project analyzes workforce utilization, billability, and capacity planning using Power BI. 
The goal is to help organizations understand resource allocation and identify over- or under-utilization.

## Tools & Technologies
- Power BI
- DAX
- Power Query
- Microsoft Excel

## Key Features
- 5 interactive dashboards
- KPIs for utilization, billability, and capacity
- Planned vs actual hours comparison
- Drill-downs and slicers for detailed analysis

## Insights Delivered
- Identified resource over-allocation and under-utilization
- Highlighted workload imbalance across teams
- Supported data-driven workforce planning decisions
